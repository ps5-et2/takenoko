# Takenoko

![Java CI with Maven](https://github.com/pns-si3-projects/projet2-ps5-20-21-takenoko-2021-et/workflows/Java%20CI%20with%20Maven/badge.svg)

## Building

Local build including tests:

```
$ mvn -B package
```

## Running

```
$ mvn exec:java
```

Requires Java 11 (JAR 55.0).
